
## ***Dummy shopping list***


<p align="left">It was a really fun experience to build a small dummy project using AI on top of React + Vite. The idea came to me quickly.

The code, though, is completely unreadable for humans: no structure, no components.

_______

### ***Features***

- Fully mobile-responsive design
- Supports 3 languages
- Light / Dark theme
- Built-in calculator
- Clean "list-only" mode
- Price tracking mode: save prices and see how they change over time
- Track total spendings
- All data saved in browser localStorage
- Option to wipe all data from localStorage
